L’objectif de ce projet est de développer une application pour la mise en place d’un graphe à colorier. Ce graphe se traduit par un réseau de processus interconnectés, créé en fonction d’une description donnée en entrée (http://cedric.cnam.fr/~porumbed/graphs/). 

## Dossier src
Il contient 3 autres dossiers parser, réseau et graph qui contiennent respectivement les fichiers de parsing du fichier DIMACS, mise en place d'un réseau de processus interconnectés et la coloration du graphe. 

## Dossier ressources 
Il contient les fichiers DIMACS

## Dossier build
Contient le Makefile

## Lancer le projet revient à lancer le fichier start.sh comme suit:
chmod +x start.sh
bash start.sh
